Preguntas Frecuentes

¿Cómo funciona el test de velocidad?

Descarga y sube archivos de prueba midiendo el tiempo transcurrido para calcular la velocidad en Mbps.

¿Es preciso el resultado?

Es una estimación aproximada basada en HTTP. Para mayor precisión usa Speedtest de Ookla.

¿Consume datos móviles?

Sí, cada prueba consume aproximadamente 5-20 MB de datos.
